    Open System Dynamics¶

    cqed_sim supports Lindblad open-system simulations via NoiseSpec, which specifies relaxation, dephasing, and thermal environments for each subsystem.

    NoiseSpec¶

    from cqed_sim.sim import NoiseSpec

noise = NoiseSpec(
    t1=30e-6,          # Qubit T₁ relaxation (s)
    tphi=15e-6,        # Qubit T_φ pure dephasing (s)
    kappa=1e4,         # Cavity decay rate (1/s)
    nth=0.01,          # Cavity thermal occupation
)

    All Parameters¶

    Parameter Type Description
    t1 float \| None Aggregate qubit T₁ relaxation (s)
    transmon_t1 tuple[float \| None, ...] \| None Per-transition T₁: (T1_ge, T1_fe, ...)
    tphi float \| None Qubit pure dephasing time T_φ (s)
    kappa float \| None Cavity/storage decay rate (1/s)
    nth float Cavity thermal photon number
    kappa_storage float \| None Storage-specific decay (defaults to kappa)
    kappa_readout float \| None Readout cavity decay (1/s)
    nth_storage float \| None Storage thermal occupation
    nth_readout float \| None Readout thermal occupation

    Basic Usage¶

    Pass noise to simulate_sequence():

    from cqed_sim.sim import simulate_sequence, SimulationConfig, NoiseSpec

noise = NoiseSpec(t1=30e-6, tphi=15e-6, kappa=1e4)

result = simulate_sequence(
    model, compiled, initial_state, drive_ops,
    config=SimulationConfig(frame=frame),
    noise=noise,
)

    When noise is present, the solver automatically uses mesolve (Lindblad master equation) instead of sesolve.

    Collapse Operators¶

    NoiseSpec generates the following Lindblad jump operators:

    Source Collapse Operator Condition
    Aggregate qubit relaxation \(\sqrt{1/T_1} \cdot b\) t1 is set, transmon_t1 is not
    Per-transition relaxation \(\sqrt{1/T_{1,j}} \cdot \|j{-}1\rangle\langle j\|\) transmon_t1 is set
    Pure dephasing (two-level) \(\sqrt{1/(2T_\phi)} \cdot \sigma_z\) tphi is set, n_tr=2
    Pure dephasing (multilevel) \(\sqrt{1/(2T_\phi)} \cdot n_q\) tphi is set, n_tr≥3
    Cavity decay \(\sqrt{\kappa(n_{\text{th}}+1)} \cdot a\) kappa is set
    Cavity thermal excitation \(\sqrt{\kappa \cdot n_{\text{th}}} \cdot a^\dagger\) kappa and nth > 0

    Multilevel Transmon Decay¶

    For multilevel transmon simulations, specify per-transition relaxation times:

    noise = NoiseSpec(
    transmon_t1=(30e-6, 15e-6),   # T1_ge = 30 µs, T1_fe = 15 µs
    tphi=15e-6,
)

    This creates separate collapse operators \(\sqrt{1/T_{1,ge}} \|g\rangle\langle e\|\) and \(\sqrt{1/T_{1,fe}} \|e\rangle\langle f\|\) instead of a single aggregate lowering operator.

    Custom Collapse Operators¶

    Pass additional collapse operators directly:

    import qutip as qt

# Custom collapse operator
custom_c_op = 0.001 * qt.destroy(model.n_cav)

result = simulate_sequence(
    model, compiled, initial_state, drive_ops,
    config=config,
    noise=noise,
    c_ops=[custom_c_op],
)

    The c_ops parameter is combined with those generated by NoiseSpec.

    Accessing Collapse Operators Directly¶

    from cqed_sim.sim import collapse_operators

c_ops = collapse_operators(model, noise)
# Returns list of qt.Qobj collapse operators

    Three-Mode Noise¶

    For three-mode systems, separate storage and readout decay:

    noise = NoiseSpec(
    t1=30e-6,
    tphi=15e-6,
    kappa_storage=1e4,
    kappa_readout=5e5,
    nth_storage=0.01,
    nth_readout=0.0,
)
